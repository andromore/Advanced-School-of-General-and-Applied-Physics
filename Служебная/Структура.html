Структура сайта:

  • Главная страница:

    • Ссылки для загрузки материалов, относящихся к данному предмету

    • Ссылки на полезные внешние ресурсы

Структура репозитория

Ссылка на сам репозиторий.

Структура репозитория имеет вид:

  • В корне лежат:
    • Папки курсов (Курс 1, Курс 2, ...)
    • Папки предметов, изучающихся более чем на одном курсе
    • Ресурсы сайта
      • index.html - Главная страница сайта
      • theme.css - Файл CSS-темы сайта
      • theme.scss - Файл SCSS-темы сайта - которая потом компилируется в theme.css с помощью BuildTheme
      • head.js - Скрипт, который должен присутствовать на всех страницах сайта - отвечает за загрузку ресурсов и глобальных функций и объектов
      • body.js - Скрипт, который должен присутствовать на всех страницах сайта - отвечает за прорисовку глобального меню, шапки сайта, низа сайта
      • BuildTheme - Скрипт компиляции SCSS-темы сайта (работает на linux при наличии пакета sassc)
    • Ресурсы репозитория
      • .gitignore - Файлы, игнорируемые git-ом при сохранении изменений
      • README.md - Описание репозитория
    • Папка "Служебная" с статьями об устройстве сайта
    • Папка "Статьи" с ознакомительными статьями о ВШОПФ
    • Папка "Фоны" с файлами изображений для фонового слайд-шоу сайта и страницей их загрузки
    • Папка "Хлам" с неотсортированными файлами
  • В папках курсов лежат:
    • Папки предметов, изучающихся только на этом курсе
    • Материалы, относящиеся к курсу в целом
    • Служебные ресурсы сайта:
      • index.html - Страница, отвечающая данному курсу (содержит ссылки на страницы предметов курса и прочую информацию)
  • В папках предметов лежат:
    • Материалы по данному предмету
    • Служебные ресурсы сайта:
      • index.html - Страница загрузки материалов предмета
