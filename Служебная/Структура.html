Добавление материалов

Будущие поколения ВШОПФа будут благодарны за твой вклад в общее дело. Узнавай как добавлять свои материалы в этом разделе и присылай их нам. Оставь свой след в истории ВШОПФ!!!

Самый простой и донельзя банальный способ - написать администратору сайта. Контакты текущего админа:

  • Вконтакте
  • Телеграм

Также можно добавить материалы в Секретные материалы. Для этого можно написать сообществу.

При использовании классического интерфейса GitHub:

  • Если требуется что-то добавить, в разделе "Discussions" создан чат "Добавить".
  • Для ваших идей в разделе "Discussions" создан чат "Идеи".
  • Для уведомления об ошибке предназначен раздел "Issues".

Структура сайта

Ссылка на сайт репозитория.

Структура сайта:

  • Главная страница:
    • Ссылки на страницы из папки служебные (различные информационные статьи)
    • Ссылки на страницы курсов
  • Страницы курсов
    • Ссылки на страницы предметов, проходимых на данном курсе, по семестрам
    • Ссылки для загрузки материалов, относящихся ко всему курсу
    • Ссылки на материалы, внутренние статьи и внешние ресурсы, относящиеся ко всему курсу в целом
  • Страницы предметов
    • Ссылки для загрузки материалов, относящихся к данному предмету
    • Ссылки на полезные внутренние статьи по данному предмету
    • Ссылки на полезные внешние ресурсы

Структура репозитория

Ссылка на сам репозиторий.

Структура репозитория имеет вид:

  • В корне лежат:
    • Папки курсов (Курс 1, Курс 2, ...)
    • Папки предметов, изучающихся более чем на одном курсе
    • Служебные ресурсы сайта:
      • index.html - Главная страница сайта
      • theme.css - Файл CSS-темы сайта
      • background.jpg - Фоновое изображение сайта
      • script.js - Скрипт, осуществляющий базовые функции
      • global.js - Скрипт, отвечающий за глобальное меню, "шапку" и "дно" сайта
      • local.js - Скрипт, отвечающий за дополнительные меню: список информационных статей и навигация по сообществу
    • Служебные ресурсы репозитория:
      • .gitignore - Файлы, игнорируемые git-ом при сохранении изменений
      • README.md - Описание репозитория
    • Папка "Служебная" с различными информационными статьями
  • В папке "Служебная" лежат:
    • Структура.html - данная статья
    • structure.js - скрипт навигации по данной статье
    • Информационные статьи
    • Ресурсы к информационным статьям
  • В папках курсов лежат:
    • Папки предметов, изучающихся только на этом курсе
    • Материалы, относящиеся к курсу в целом
    • Служебные ресурсы сайта:
      • index.html - Страница, отвечающая данному курсу (содержит ссылки на страницы предметов курса и прочую информацию)
      • local.js - Скрипт, отвечающий за отображение локальных навигационных меню: меню предметов по каждому семестру в отдельности
  • В папках предметов лежат:
    • Материалы по данному предмету
    • Служебные ресурсы сайта:
      • index.html - Страница загрузки материалов предмета
      • local.js - Скрипт, отвечающий за отображение локальных навигационных меню по конкретному предмету

Материалами считаются:

  • Документы (pdf, djvu, tex)
  • Картинки (png, jpg, svg)
  • Ссылки на другие ресурсы

(GitHub ограничивает размер файлов: предел 25 MB)

Разработка сайта

Раздел создан для желающих помочь проекту не словами, а делами. Читайте и присоединяйтесь к команде.

Сайт разрабатывается на голом HTML5/CSS3 с применением JavaScript во избежание старого-доброго копирования, располагается непосредственно в репозитории GitHub, поверх которого и был написан, который выступает в качестве хранилища материалов, и хостится с использованием технологии GitHub Pages.

В файле script.js создаются элементы <site-header>, <site-head>, <site-body>, <site-main>, <site-left-bar>, <site-nav>, <site-content>, <site-right-bar>, <site-footer>, <site-foot>. Они выполняют функцию упрощения вёрстки сайта. Здесь же создаётся класс Site, который упрощает управление сайтом.

Таким образом тело сайта принимает вид:

<site-header>
    <site-head>
        Здесь находится шапка сайта
    </site-head>
</site-header>
<site-body>
    <site-main>
        <site-left-bar>
            <site-nav>
                Здесь находятся навигационные меню
            </site-nav>
            На данный момент блок используется
            только для навигационных меню
        </site-left-bar>
        <site-content>
            Здесь находится основной код страницы
        </site-content>
        <site-right-bar>
            На данный момент блок не используется
        </site-right-bar>
    </site-main>
</site-body>
<site-footer>
    <site-foot>
        Здесь находится дно сайта
    </site-foot>
</site-footer>
<script src="./script.js"></script> - Загружает функционал блоков
<script src="./global.js"></script> - Инициализация страницы
<script src="./local.js"></script> - Локальные навигационные меню

Скрипт script.js производит загрузку функциональности блоков и класса Site и сам по себе ничего не делает.

Скрипт global.js производит действия, относящиеся к сайту в целом. Конкретно:

  • Инициализация класса Site
  • Загрузка темы сайта theme.css
  • Генерация глобального навигационного меню
  • Генерация содержимого <site-head> - шапки сайта
  • Генерация содержимого <site-foot> - дна сайта

Последние два пункта выполняются (т.к. не было придумано ничего лучше) путём загрузки содержимого соответствующих блоков, банально прописанного непосредственно внутри скрипта.
