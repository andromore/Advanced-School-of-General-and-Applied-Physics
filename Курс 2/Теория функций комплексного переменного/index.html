О предмете

Лекционные и практические занятия по данному предмету проводятся одним преподавателем вместе в установленном им порядке.

В течение семестра по данному предмету предусмотрены 4 контрольные работы по темам курса, соответственно:

  1. Введение (арифметика комплексных чисел и основы комплексного анализа)
  2. Камфорные отображения
  3. Ряд Лорана (и связанные темы)
  4. Операционное исчисление (в т.ч. решение дифференциальных уравнений этим методом)

Задания из контрольных оцениваются по четырёхбалльной шкале: минус, минус-плюс, плюс-минус и, соответственно, полюс.

В случае, если студент написал контрольную не удовлетворительно, он будет решать одно задание из этой контрольной на экзамене. Критерии спрашивайте у преподавателя перед контрольной. Таким образом, по результату работы в семестре, любой студент может получить 4 дополнительных вопроса на экзамене.

Каждый студент должен сдать все задачи с контрольных на плюс-минус и выше - это условие допуска к экзамену.

Если студент не закрыл задачу на контрольной - он отправляется на досдачи, на которых он должен будет сдать решённую задачу (сделать это надо заранее) и ответить на вопросы по этой теме.

Экзамен

На экзамене каждый студент получает билет, в котором содержатся 2 теоретических вопроса и 4 задачи, из которых студент в обязательном порядке пишет теорию и только те задачи, темы которых соответствуют неудовлетворительно написанным контрольным.

Студент допускается до экзамена только после того, как все задачи будут закрыты на плюс-минус и выше (закрытие задачи на досдаче не влияет на решаемые на экзамене задачи).
