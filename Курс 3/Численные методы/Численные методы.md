# Численные методы
## РГР №1
Найти корень уравнения $\dfrac{sin(x^2)}{x} = a$ при $a = 1/2$.

## РГР №2
Найти минимум функции $f(x, y) = (xy - cos(x))^2 + sin^2(xy)$.

## РГР №3
Построить интерполяцию по 7 точкам и аппроксимацию зависимости корня $x_*(a)$ от параметра $a$.

## РГР №4
Найти первообразную (неопределённый интеграл) зависимости корня $x_*(a)$ от параметра $a$. Оценить максимальное число точек для точности $10^{-5}$ при $a \in [0, 1]$.

## РГР №5
Найти производную $dx_*/da$ вт очке $a = 1/2$ и оценить максимальную точность.

## РГР №6
Найти решение обыкновенного дифференциального уравнения $y'' - y + y^3 = 0$ с заданными начальными условиями $y'(0) = 0$, $y(0) = 2$.

## РГР №7
Найти решение параболического уравнения $u_t = u_{xx}$ при заданных граничных условиях $u(-1, t) = u(1, t) = 1$ и начальных условиях $u(x, 0) = \delta(x)$. По возможности сравнить с точным решением или проверить сохранение интегралов. В ответе должен быть график функции от $x$  $t$.
