Куда я попал?

Это неофициальный репозиторий (облачное хранилище на GitHub) Высшей Школы Общей и Прикладной Физики - одного из факультетов ННГУ им. Лобачевского. Данный репозиторий создан для публикации материалов, связанных с обучением на нашем факультете. Таковыми материалами считаются:

  • Учебники, задачники, методички, шпаргалки по предметам
  • Задания и решения контрольных работ
  • Вопросы, экзаменационные билеты и задачи

Также, с использованием технологии GitHub Pages поверх репозитория построен сайт. Таким образом материалы представляются в виде более привычном большинству пользователей (сайт с ссылками для скачивания материалов и статьями-инструкциями). Для желающих ознакомиться с факультетом ВШОПФ или с устройством сайта ссылки ниже.

Статьи о ВШОПФ Устройство проекта

Данный сайт ставит своей целью помочь найти полезные материалы и разобраться, поэтому настоятельно не рекомендуется его использование для бездумного списывания.

Бакалавриат

Программа бакалавриата предполагает прохождение четырёх курсов. По данным опросов психологов, старшекурсники приписывают наибольшую сложность третьему курсу, причём считается, что с первого курса сложность увеличивается экспоненциально.
