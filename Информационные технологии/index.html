Теория

В данном разделе будут приведены ссылки на сайты и книги для изучения некоторых полезных языков программирования. На ВШОПФ таковыми являются C/C++ и Python.

Непосредственно на парах изучаются два языка программирования: C и C++, причём второй является фактически расширением первого, поэтому в C++ вы можете использовать всё, что знаете из C с некоторыми оговорками.

Третий язык программирования - Python - востребован по следующим причинам:

  • Его легко изучить даже самостоятельно и без опыта
  • Хорош для "не программистов": там нет переполнения, строгой типизации и прочих "плюшек"
  • Подробный и понятный вывод ошибок - интерпретатор напишет вам в какой строчке ошибка, тип ошибки, и т.д.
  • Имеет большое количество готовых библиотек - многое уже готово, достаточно просто найти
  • "Из коробки" умеет работать с комплексными числами и т.д.

Ввиду указанных выше причин, он используется многими учёными в ИПФ РАН, поэтому вы с высокой долей вероятности столкнётесь с ним на учебной практике или при любой другой работе в ИПФ РАН.

Библиотеки, которые вам могут пригодиться.

NumPy MatPlotLib Pandas SciPy SymPy

Здесь:

  • NumPy - библиотека с мощными инструментами для линейной алгебры и прочего;
  • MatPlotLib - библиотека для визуализации двух и трёхмерных данных, которая умеет работать с NumPy;
  • Pandas - библиотека для работы с таблицами (не хуже чем Microsoft Excel и другие) и временными рядами;
  • SciPy - библиотека, позволяющая интегрировать, искать минимумы/максимумы функций, решать дифференциальные уравнения и т.д.;
  • SymPy - библиотека для работы с символьными вычислениями: интегрирование, дифференцирование и прочие манипуляции с функциями.

Данные библиотеки не входят в стандартную библиотеку и требуют отдельной установки через пакетный менеджер pip.

Практика

1. Численное интегрирование

Задача: Имеется набор функций (каждый выбирает функции сам, но функции вбиваются в массив внутри программы и пользователю предоставляется выбор одной из них), требуется проинтегрировать выбранную пользователем функцию численно (Интегрирование производится на некотором отрезке с некоторым задаваемым шагом).

Пользовательский интерфейс: Сначала предлагается выбрать (из списка) функцию для интегрирования, после чего ожидается ввод начала отрезка интегрирования, конца отрезка интегрирования и шаг интегрирования. После ввода данных пользователь получает численное значение интеграла от выбранной функции.

Методы численного интегрирования определённых интегралов:

  • Метод прямоугольников (аналогичен трапециям)
  • Метод трапеций - предлагаемый метод
  • Метод Монте-Карло

2. Метод Гаусса

Задача: Решить систему линейных дифференциальных уравнений.
(Изначально разрешалось не учитывать варианты, когда система имеет бесконечно много решений или не имеет их вовсе)

Дополнительные задания:

  • Научиться генерировать коэффициенты система автоматически
  • Посчитать погрешность метода (для этого генерируется решение системы, генерируются её коэффициенты, по решениям и коэффициентам считаются свободные члены, чтобы данные решения удовлетворяли системе, после чего полученная система решается, а полученное решение сравнивается с оригинальным)
